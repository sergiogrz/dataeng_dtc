## Requirements
* System requirements:
    + docker
    + docker-compose
    + python >= 3.9
    + terraform
    + google cloud SDK
* Python package requirements:
    + notebook
    + pgcli
    + psycopg2-binary
    + SQLAlchemy
    + pandas
